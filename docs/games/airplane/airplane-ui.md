# 飞机游戏界面设计文档

## 1. 文档目的

本文档定义 `飞机` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 竖屏优先
- 游戏画面占据主体
- 信息轻量化，不遮挡视野

## 2. 总体布局

竖屏结构：

```text
┌──────────────────────────┐
│ 暂停    关卡:1    分数    │
├──────────────────────────┤
│                          │
│                          │
│      游戏画面区域         │
│   (星空背景+敌机+子弹)    │
│                          │
│                          │
│      玩家飞机             │
│                          │
├──────────────────────────┤
│ HP: ♥♥♥  火力: ★★★      │
│ 道具槽: [护盾] [炸弹]     │
└──────────────────────────┘
```

## 3. 区块设计

### 3.1 顶部HUD

- 暂停按钮
- 当前关卡
- 当前分数
- 半透明背景，不遮挡游戏画面

### 3.2 游戏画面

- 深空背景，星空/云层滚动视差
- 敌机从上方出现
- 子弹轨迹清晰
- 爆炸粒子特效

### 3.3 底部HUD

- 生命值（心形图标）
- 火力等级（星星图标）
- 道具槽

## 4. 视觉方向

### 4.1 背景

- 深空渐变色 `#0A0E27` → `#1A1A4E`
- 星空粒子缓慢移动
- 下方云层半透明滚动

### 4.2 角色设计

- 玩家飞机：银灰色/青色，简洁几何造型
- 小敌机：暗红色三角形
- 中敌机：橙色菱形
- 大敌机：紫色大型造型
- Boss：深红色大型造型，独立血条

### 4.3 特效

- 子弹：亮色细线/小圆点
- 爆炸：粒子扩散，红橙黄渐变
- 道具：发光浮动图标

## 5. 色彩建议

- 背景：`#0A0E27` → `#1A1A4E`
- 玩家飞机：`#00D68F`（青绿）
- 敌机：`#FF4757`（红）/ `#FFA534`（橙）
- 子弹：`#2BD0A5`（青）
- 爆炸：`#FF6B6B`（红橙）

## 6. 状态设计

- 游戏中
- 暂停
- 游戏结束
- 关卡过渡

## 7. 组件拆分

- `AirplaneWarGameArea`
- `PlayerPlane`
- `EnemyPlane`
- `Bullet`
- `ExplosionEffect`
- `AirplaneWarHUD`
- `AirplaneWarResultOverlay`

## 8. 验收标准

- 游戏画面流畅（60fps）
- 触屏/键盘操作响应即时
- 敌机/子弹/道具清晰可辨
- 爆炸特效有反馈感
- HUD不遮挡游戏画面
- 暂停/恢复不打断游戏节奏
