# 扫雷游戏界面设计文档

## 1. 文档目的

本文档定义 `扫雷` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 网格清晰，格子辨识度高
- 数字颜色区分明显
- 操作便捷（点击/长按）

## 2. 总体布局

竖屏结构：

```text
┌──────────────────────────┐
│ 返回    扫雷    重新开始   │
│ 🚩 xxx    ⏱ xxx         │
├──────────────────────────┤
│                          │
│      扫雷网格             │
│   (可滚动/缩放)           │
│                          │
│                          │
│                          │
├──────────────────────────┤
│   点击翻开 / 长按标记     │
└──────────────────────────┘
```

## 3. 区块设计

### 3.1 顶部状态栏

- 剩余雷数（旗子图标 + 数字）
- 计时器
- 重新开始按钮

### 3.2 网格区域

- 自适应网格大小
- 高级模式支持滚动
- 格子有轻微凸起/凹陷的立体感

### 3.3 底部提示

- 操作提示文字

## 4. 格子视觉

| 格子状态 | 表现 |
|----------|------|
| 未翻开 | 浅色凸起方块，有立体感 |
| 已翻开 | 深色凹陷方块 |
| 数字1 | 蓝色 `#2196F3` |
| 数字2 | 绿色 `#4CAF50` |
| 数字3 | 红色 `#F44336` |
| 数字4 | 深蓝 `#1A237E` |
| 数字5 | 暗红 `#880E4F` |
| 数字6 | 青色 `#00838F` |
| 数字7 | 黑色 `#1A1A2E` |
| 数字8 | 灰色 `#757575` |
| 雷 | 深色底 + 炸弹图标 |
| 旗子 | 红色三角旗 |

## 5. 色彩建议

- 背景：`#0D1441`
- 未翻开：`#2D3A7E`
- 已翻开：`#1A225D`
- 数字色：见上方

## 6. 状态设计

- 准备中（首次点击前）
- 游戏中
- 胜利
- 失败

## 7. 组件拆分

- `MineGrid`
- `MineCell`
- `MineCounter`
- `MineTimer`
- `MineResultOverlay`

## 8. 验收标准

- 格子清晰可辨，大小适中可点击
- 数字颜色区分明显
- 首次点击安全
- 0格子自动展开
- 旗子标记清晰
- 失败时显示所有雷
- 胜利时展示结果和时间
