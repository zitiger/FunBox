## 扫雷游戏规则

### 1. 游戏目标
在不触雷的情况下，翻开所有非雷格子，根据数字推理出所有雷的位置。

### 2. 难度选择
- 初级：9×9 网格，10颗雷
- 中级：16×16 网格，40颗雷
- 高级：16×30 网格，99颗雷

### 3. 游戏盘面
- 每个格子初始为未翻开状态
- 格子下方可能隐藏着雷或数字（0-8）

### 4. 操作规则
- **点击**：翻开格子
  - 是雷 → 游戏失败
  - 是数字 → 显示数字
  - 数字为0 → 自动翻开周围所有相邻格子（连锁展开）
- **长按/右键**：标记/取消标记
  - 循环切换：无标记 → 旗子 → 问号 → 无标记

### 5. 首次安全
- 首次点击的格子不会是雷
- 首次点击位置周围8格也不会是雷
- 雷在首次点击后才生成

### 6. 数字含义
- 数字表示该格子周围8个相邻格子中隐藏的雷数
- 例如：数字"3"表示周围8格中有3颗雷

### 7. 胜利与失败
- **胜利**：所有非雷格子被翻开
- **失败**：点击到雷（所有雷显示，游戏结束）

### 8. 计时
- 从首次点击开始计时
- 记录完成时间作为成绩
