# 俄罗斯方块游戏界面设计文档

## 1. 文档目的

本文档定义 `俄罗斯方块` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 竖屏优先
- 盘面是视觉主角
- 7种方块颜色鲜明区分
- 操作便捷

## 2. 总体布局

竖屏结构：

```text
┌──────────────────────────────┐
│ 暂停    俄罗斯方块    分数    │
├──────────────┬───────────────┤
│              │  下一块        │
│              │  [预览]        │
│   10×20      │               │
│   盘面       │  等级: 5       │
│              │  行数: 42      │
│              │               │
│              │               │
│              │               │
├──────────────┴───────────────┤
│  [←] [↓] [→] [旋转] [硬降]   │
└──────────────────────────────┘
```

## 3. 区块设计

### 3.1 盘面区

- 10×20 网格，深色底板
- 网格线隐约可见
- 已锁定方块显示颜色+边框
- 当前方块显示颜色+轻微发光
- 幽灵方块（半透明预览硬降位置）

### 3.2 右侧信息区

- 下一块预览
- 当前等级
- 已消行数
- 最高分

### 3.3 底部操作区

- 方向键：左/下/右
- 旋转按钮
- 硬降按钮

## 4. 方块色彩

| 方块 | 颜色 | 色值 |
|------|------|------|
| I | 青色 | `#00D4FF` |
| O | 黄色 | `#FFD700` |
| T | 紫色 | `#9B59B6` |
| S | 绿色 | `#2ECC71` |
| Z | 红色 | `#E74C3C` |
| J | 蓝色 | `#3498DB` |
| L | 橙色 | `#E67E22` |

## 5. 整体色彩

- 背景：`#0D1441`
- 盘面底板：`#1A225D`
- 网格线：`#242D6E`（隐约可见）
- 方块：见上方色表

## 6. 状态设计

- 游戏中
- 暂停
- 消行动画
- 等级提升
- 游戏结束

## 7. 组件拆分

- `TetrisBoard`
- `TetrisBlock`
- `NextPiecePreview`
- `TetrisScorePanel`
- `TetrisControlBar`
- `TetrisResultOverlay`

## 8. 验收标准

- 7种方块颜色鲜明区分
- 盘面网格清晰
- 下落/旋转/消行动画流畅
- 操作按钮响应即时
- 下一块预览正确
- 信息面板信息清晰
- 竖屏横屏均适配
