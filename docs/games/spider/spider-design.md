# 蜘蛛平台接入设计文档

## 1. 概述

在 FunBox 平台中实现经典的蜘蛛纸牌（Spider Solitaire）小游戏模块。
- **模式**：单人
- **用牌**：2副标准52张扑克牌（共104张，不含大小王）
- **难度**：单花色（入门）、双花色（中级）、四花色（高级）
- **目标**：将所有牌按K到A的递减序列从牌列中移除

## 3. 数据结构

### 3.1 游戏会话 (SpiderSession)

```dart
class SpiderSession {
  List<List<Card>> columns;       // 10列牌列
  List<List<Card>> dealBuffer;    // 剩余待发牌（按批）
  int difficulty;                 // 难度：1=单花色, 2=双花色, 4=四花色
  int score;                      // 当前分数
  int moveCount;                  // 移动次数
  int completedSets;              // 已完成套数
  bool isGameOver;                // 是否结束
  bool isWin;                     // 是否获胜
  int startTimeMs;                // 开始时间
  int elapsedMs;                  // 已用时间
}
```

## 4. 游戏流程

```
开始
  ├─ 选择难度（单花色/双花色/四花色）
  ├─ 洗牌 104 张（2副牌）
  ├─ 10列牌列：前4列各6张，后6列各5张（共54张）
  ├─ 每列仅顶部1张翻开
  ├─ 剩余50张分5批，每批10张
  └─ 进入操作循环：
       ├─ 玩家拖拽牌 → 按规则验证移动
       ├─ 完成K→A递减序列 → 自动移除
       ├─ 点击发牌 → 10列各加1张（需所有列非空）
       └─ 所有牌移除 → 胜利
```

## 5. 移动规则

| 规则 | 说明 |
|------|------|
| 牌列移动 | 可将一张或多张牌移动到另一列，目标列顶部牌必须比移动牌点数大1（花色不限） |
| 递减序列 | 同列中翻开牌必须点数递减 |
| 完整序列 | K→A递减序列自动移除到左下角 |
| 空列 | 任何牌都可放入空列 |
| 发牌 | 点击发牌条给10列各加1张，需所有列非空 |

## 6. 计分规则

| 操作 | 得分 |
|------|------|
| 初始分值 | 500 |
| 每移动一次 | -1 |
| 完成一组K→A | +100 |

## 8. 渲染方案

| 元素 | 渲染方式 |
|------|----------|
| 牌面 | 白底 + 角标点数 + 中央花色 |
| 牌背 | 深蓝底 + 网格纹理 |
| 花色色值 | ♥♦ 红色，♠♣ 黑色 |
| 选中态 | 牌面上移 + 金色边框 |
| 已完成序列 | 缩小版牌组展示在左下角 |
