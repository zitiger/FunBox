# 蜘蛛游戏界面设计文档

## 1. 文档目的

本文档定义 `蜘蛛` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 横屏优先，10列牌列清晰可见
- 拖拽操作流畅
- 已完成序列进度一目了然

## 2. 总体布局

横屏结构：

```text
┌──────────────────────────────────────────────────────┐
│ 返回           蜘蛛纸牌      分数        重新开始       │
├──────────────────────────────────────────────────────┤
│ 列1  列2  列3  列4  列5  列6  列7  列8  列9  列10      │
│                                                      │
│                                                      │
│                                                      │
│                                                      │
│                                                      │
│                                                      │
├──────────────────────────────────────────────────────┤
│ [发牌: 剩余5批]          [已完成: 0/8组]               │
└──────────────────────────────────────────────────────┘
```

## 3. 区块设计

### 3.1 顶部状态栏

- 返回按钮
- 当前分数
- 重新开始按钮

### 3.2 牌列区

- 10列等宽排列
- 扣置牌显示牌背
- 翻开的牌可拖拽
- 支持多张牌同时选定

### 3.3 底部信息栏

- 发牌条：显示剩余批次数，点击发牌
- 已完成序列：显示已完成组数

## 4. 牌面表现

- 白底牌面，角标点数+花色
- 牌背：深蓝底+网格纹理
- 选中态：牌面上移+金色边框
- 已完成序列：缩小版牌组

## 5. 色彩建议

- 背景：`#0D1441`
- 牌面：白色
- 牌背：`#1A225D`
- 强调色：`#FFA534`

## 6. 状态设计

- 游戏中
- 发牌中
- 序列移除中
- 胜利

## 7. 组件拆分

- `SpiderTableShell`
- `SpiderCardWidget`
- `SpiderColumnWidget`
- `SpiderDealBar`
- `SpiderCompletedSets`
- `SpiderResultOverlay`

## 8. 验收标准

- 10列牌列正确显示
- 拖拽移动流畅
- 完成序列自动移除
- 发牌条功能正常
- 已完成的序列清晰可见
- 胜利时展示结果
