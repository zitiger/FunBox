# 扑克游戏界面设计文档

## 1. 文档目的

本文档定义 `扑克` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 牌桌优先，公共牌居中
- 横屏为主
- 筹码信息清晰

## 2. 总体布局

横屏"牌桌"结构：

```text
┌──────────────────────────────────────────┐
│ 顶部状态栏（返回/设置）                     │
├────────────────────┬─────────────────────┤
│  AI1 区域           │  AI2 区域           │
│  筹码/手牌背        │  筹码/手牌背         │
├────────────────────┴─────────────────────┤
│         公共牌区域（5张）                   │
│         底池: xxx                         │
├──────────────────────────────────────────┤
│  玩家手牌（2张）   筹码: xxx              │
│  [弃牌] [跟注] [加注]                     │
└──────────────────────────────────────────┘
```

## 3. 区块设计

### 3.1 公共牌区

- 5张牌居中排列，横向展开
- 底池金额在牌下方显示
- 翻牌/转牌/河牌逐步揭示

### 3.2 玩家手牌区

- 底部2张底牌正面朝上
- 筹码余额显示
- 操作按钮在牌下方

### 3.3 AI 对手区

- 手牌背面显示
- 筹码余额显示
- 当前操作高亮

## 4. 牌面表现

- 白底牌面，角标点数+花色
- 红心/方块红色，黑桃/梅花黑色
- 选中态：牌面上移+金色边框

## 5. 色彩建议

- 背景：`#0D1441`（深海蓝）
- 牌桌底色：`#1A225D`
- 牌面：白色
- 筹码：`#FFA534`（暖橙）
- 公共牌区：`#242D6E`

## 6. 状态设计

- 等待下注
- 翻牌中
- 转牌中
- 河牌中
- 摊牌比牌
- 游戏结束

## 7. 组件拆分

- `PokerTableShell`
- `PokerCardWidget`
- `PokerPlayerPanel`
- `PokerCommunityCards`
- `PokerActionBar`
- `PokerChipWidget`
- `PokerResultOverlay`

## 8. 验收标准

- 公共牌和底牌清晰可辨
- 筹码信息实时更新
- 操作按钮状态正确
- 摊牌时牌型对比直观
- 适配横屏和竖屏
