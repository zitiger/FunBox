# 贪吃蛇游戏界面设计文档

## 1. 文档目的

本文档定义 `贪吃蛇` 在 `FunBox` 中的主游戏界面布局与视觉层级，用于后续 Flutter 页面实现。

设计目标：
- 优先服务"快速上手"
- 竖屏优先
- 盘面优先，信息轻量化
- 保持真实产品感

## 2. 设计前提

- 屏幕方向：竖屏为主
- 设计重心：蛇盘面优先
- 遵循 `docs/game-ui-style-requirements.md` 的整体风格要求

## 3. 页面目标

对局主界面要在 1 秒内回答用户三个问题：
- 蛇在哪里
- 食物在哪里
- 分数多少

## 4. 总体布局

### 4.1 布局结构

竖屏采用"顶部信息 + 盘面 + 底部操作"结构：

```text
┌──────────────────────────┐
│ 顶部状态栏（返回/暂停）    │
│ 分数区（分数/最高分）      │
├──────────────────────────┤
│                          │
│      游戏盘面区域         │
│    (网格+蛇+食物)         │
│                          │
├──────────────────────────┤
│    方向键控制区           │
│   (或滑动操作引导)        │
└──────────────────────────┘
```

## 5. 区块设计

### 5.1 顶部状态栏

- 返回按钮（左上）
- 暂停按钮（右上）
- 游戏标题居中

### 5.2 分数区

- 当前分数与最高分并列显示
- 采用小卡片承载，统一圆角与阴影

### 5.3 盘面区域

- 网格底板：深色，与背景形成对比
- 蛇身：渐变色圆角方块，头部带眼睛标识方向
- 食物：红色圆形，带微光呼吸效果
- 空格可见但不喧宾夺主

### 5.4 底部操作区

- 竖屏时显示方向键（上/下/左/右）
- 方向键采用轻拟物风格
- 横屏时可由滑动替代

## 6. 视觉方向

### 6.1 背景

深色或中深色背景基底，营造"桌面感"。

### 6.2 色彩建议

- 背景深色：`#0D1441`
- 网格底板：`#1A225D`
- 蛇头：`#2BD0A5`（青绿）
- 蛇身：`#00D68F` 到 `#0B8A5E` 渐变
- 食物：`#FF6B6B`（暖红）
- 强调色：`#FFA534`（暖橙）

## 7. 状态设计

- 准备中
- 游戏中
- 暂停
- 游戏结束

## 8. 交互原则

- 方向控制优先于信息展示
- 操作反馈即时（不延迟）
- 暂停/恢复不打断游戏节奏

## 9. 组件拆分建议

- `SnakeTableShell`
- `SnakeScorePanel`
- `SnakeBoard`
- `SnakeDirectionPad`
- `SnakeResultOverlay`

## 10. 验收标准

- 用户 1 秒内能看出蛇的位置和方向
- 食物位置清晰可见
- 方向键/滑动操作响应及时
- 游戏结束时有明确的结果展示
- 页面整体像真实产品，不像活动页
