## 接龙游戏规则

### 1. 游戏目标
将一副52张扑克牌按花色从A到K依次排列到4个收集区中。

### 2. 用牌与发牌
- 使用一副标准52张扑克牌（不含大小王）
- 牌列7列：第1列1张，第2列2张，…，第7列7张
- 每列仅顶部1张牌正面朝上，其余扣置
- 剩余牌放入牌堆

### 3. 牌列移动规则
- 牌列中翻开牌必须红黑交替、点数递减
- 可将一张或多张牌移动到另一列，目标列顶部牌必须花色不同且点数大1
- 例如：红心6可放在黑桃7或梅花7下方

### 4. 空列规则
- 空列只能放置K（或K开头的序列）

### 5. 收集区规则
- 4个收集区分别对应4种花色
- 从A开始，必须按同花色递增顺序排列（A→2→3→…→K）
- 牌可以移回牌列，但需扣分

### 6. 牌堆与废牌堆
- 点击牌堆翻1张到废牌堆
- 废牌堆顶牌可移动到牌列或收集区
- 牌堆耗尽后可将废牌堆整体翻回牌堆循环使用

### 7. 计分
- 牌从废牌堆移到牌列：+5分
- 牌从牌列移到收集区：+10分
- 翻开牌列中扣着的牌：+5分
- 从收集区移回牌列：-15分

### 8. 胜利条件
- 4个收集区全部完成（各13张，共52张）→ 胜利
