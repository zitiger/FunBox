# 接龙游戏界面设计文档

## 1. 文档目的

本文档定义 `接龙` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 横屏优先，7列牌列清晰可见
- 拖拽和点击操作流畅
- 收集区进度一目了然

## 3. 区块设计

### 3.1 顶部功能区

- 牌堆（点击翻牌）
- 废牌堆（显示翻出的牌）
- 4个收集区（空位显示花色轮廓，完成度一目了然）

### 3.2 牌列区

- 7列等宽排列
- 扣置牌显示牌背
- 翻开牌可拖拽移动
- 支持多张牌同时选定

### 3.3 交互方式

- 点击牌堆：翻牌
- 点击废牌堆顶牌：自动尝试移到收集区
- 拖拽牌列中的牌：按规则移动
- 双击牌：自动尝试移到收集区

## 4. 牌面表现

- 白底牌面，角标点数+花色
- 红心/方块红色，黑桃/梅花黑色
- 牌背：深蓝底+网格纹理
- 选中态：牌面上移+金色边框

## 5. 色彩建议

- 背景：`#0D1441`
- 牌面：白色
- 牌背：`#1A225D`
- 收集区空位：半透明边框+花色轮廓
- 强调色：`#FFA534`

## 6. 状态设计

- 游戏中
- 牌堆耗尽
- 胜利（收集区全满）
- 无路可走（无法继续移动）

- 点击牌堆翻牌正常
- 双击自动收集到收集区
