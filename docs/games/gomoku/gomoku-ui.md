# 五子棋游戏界面设计文档

## 1. 文档目的

本文档定义 `五子棋` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 棋盘始终是视觉主角
- 双方落子清晰可辨
- 竖屏优先，横屏适配

## 2. 页面目标

- 当前轮到谁落子一目了然
- 最后落子位置清晰标记
- 胜负结果快速呈现

## 3. 总体布局

竖屏结构：

```text
┌──────────────────────────┐
│ 顶部状态栏（返回/重开）    │
│ 当前回合 + 双方胜场       │
├──────────────────────────┤
│                          │
│      15×15 棋盘          │
│      (正方形区域)         │
│                          │
├──────────────────────────┤
│  底部状态提示             │
└──────────────────────────┘
```

## 4. 区块设计

### 4.1 顶部状态栏

- 返回按钮（左上）
- 重新开始（右上）
- 当前回合指示（中间）

### 4.2 棋盘视觉

- 木纹底色：`#D4A574`
- 网格线：深棕色细线
- 天元/星位：小圆点标记
- 棋盘四边留白保证触控精度

### 4.3 棋子

- 黑子：`#1A1A2E`，带轻微高光
- 白子：`#F7F4F0`，带轻微阴影
- 最后落子：红点标记
- 获胜连线：金色高亮 `#FFA534`

## 5. 状态设计

- 轮到黑方 / 轮到白方
- 游戏结束（黑胜/白胜/平局）
- 棋盘交互态（可落子/等待中）

## 6. 色彩建议

- 棋盘底色：`#D4A574`（木纹暖色）
- 背景：`#0D1441`（深海蓝）
- 黑子：`#1A1A2E`
- 白子：`#F7F4F0`
- 强调色：`#FFA534`（暖橙）

## 7. 组件拆分

- `GomokuBoard`
- `StoneWidget`
- `GomokuStatusBar`
- `GomokuResultOverlay`

## 8. 验收标准

- 棋盘清晰可辨，交叉点可精准点击
- 棋子有立体感
- 最后落子和获胜连线有明确标记
- 胜负结果展示清晰
- 横竖屏棋盘自适应正方形
