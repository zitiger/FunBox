# 21点游戏界面设计文档

## 1. 文档目的

本文档定义 `21点` 在 `FunBox` 中的主游戏界面布局与视觉层级。

设计目标：
- 竖屏优先
- 牌面清晰，点数一目了然
- 操作按钮便捷

## 2. 总体布局

竖屏结构：

```text
┌──────────────────────────┐
│ 返回    21点    筹码      │
├──────────────────────────┤
│                          │
│      庄家手牌             │
│   [牌1] [牌2(暗)]        │
│   庄家点数: ?            │
│                          │
│      牌桌区域             │
│                          │
│      玩家手牌             │
│   [牌1] [牌2] [牌3]      │
│   玩家点数: 18           │
│                          │
│   下注: xxx              │
│  [要牌] [停牌] [双倍]    │
└──────────────────────────┘
```

## 3. 区块设计

### 3.1 庄家区域

- 手牌横向排列
- 第二张牌暗牌显示牌背
- 点数显示（暗牌时不显示/显示"?"）

### 3.2 玩家区域

- 手牌横向排列，全部明牌
- 点数实时更新显示
- A的计点自动取最优值

### 3.3 操作区

- 下注调节器（+/- 筹码）
- 要牌/停牌/双倍按钮
- 结果展示

## 4. 牌面表现

- 白底牌面，角标点数+花色
- 红心/方块红色，黑桃/梅花黑色
- 牌背：深蓝底+网格纹理

## 5. 色彩建议

- 背景：`#0D1441`
- 牌桌底色：`#1A225D`
- 牌面：白色
- 筹码：`#FFA534`
- 点数：白色大字

## 6. 状态设计

- 下注中
- 发牌中
- 玩家回合
- 庄家回合
- 结果展示（胜/负/平/黑杰克）

## 7. 组件拆分

- `BlackjackTableShell`
- `BlackjackCardWidget`
- `BlackjackHandWidget`
- `BlackjackChipWidget`
- `BlackjackActionBar`
- `BlackjackResultOverlay`

## 8. 验收标准

- 双方手牌清晰可见
- 点数实时更新
- 操作按钮状态正确
- 结果展示清晰
- 竖屏横屏均适配
