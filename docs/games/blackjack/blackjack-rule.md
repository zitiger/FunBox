## 21点游戏规则

### 1. 游戏目标
使手牌点数尽可能接近21点，但不超过21点，且超过庄家的点数。

### 2. 用牌
- 6副标准52张扑克牌（共312张），不含大小王
- 每次开局前洗牌，牌堆耗尽时重新洗牌

### 3. 点数计算
- 2-10：按面值计算
- J/Q/K：计10点
- A：可计1点或11点，取对玩家有利的值

### 4. 黑杰克（Blackjack）
- 初始两张牌总点数恰好21点（一张A + 一张10/J/Q/K）
- 玩家黑杰克 → 立即获胜，赔率1.5倍
- 庄家黑杰克 → 玩家立即输
- 双方黑杰克 → 平局（Push），退回赌注

### 5. 玩家操作
- **要牌（Hit）**：抽一张牌，可多次要牌
- **停牌（Stand）**：结束抽牌，等待庄家
- **双倍（Double）**：筹码翻倍，只抽一张牌后停牌（只能在首两张牌时）
- **分牌（Split）**：首两张牌点数相同时，可分为两手独立进行（首版可选支持）

### 6. 庄家规则
- 庄家点数 < 17：必须抽牌
- 庄家点数 ≥ 17：停牌
- 庄家软17（A+6）停牌

### 7. 胜负判定
- 玩家爆牌（>21）：玩家输
- 庄家爆牌（>21）：玩家赢
- 双方均未爆牌：点数大者胜
- 点数相同：平局

### 8. 赔率
- 普通胜：1:1
- 黑杰克胜：3:2（1.5倍）
- 平局：退回赌注
